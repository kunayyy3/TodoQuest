# todoquest

> 게임처럼 재밌게 투두 리스트를! RPG형 할 일 관리 브라우저 게임

## 소개

당신은 현실 속 과제, 공부, 운동 3가지 할 일을 퀘스트로 등록할 수 있습니다. 할 일을 완료하면 용사의 능력치가 올라갑니다. 강해진 용사는 던전을 공략해 마을을 발전시킵니다.

---

## 파이프라인

현실의 행동이 게임 속 성장으로 그대로 이어집니다.

---

할 일 완료 → 용사 성장 → 던전 공략 → 골드 획득 → 마을 발전 → 추가 버프

사용자는 반복적인 일상 속에서 특별한 보상을 받을 수 있습니다.

---

## 주요 기능

### 투두 시스템

* 할 일 등록
* 할 일 완료 처리
* 퀘스트 보상 획득
* 진행 현황 관리

### 용사 성장

* 3가지 능력치 성장

### 랜덤 던전

* 하루에 3개 무작위 던전 생성
* 난이도별 도전(쉬움, 보통, 어려움)
* 성장 상태에 따른 성공 판정
* 보상 획득

### 마을 발전

* 골드 사용
* 건물 업그레이드
* 마을 등급 상승
* 성장 보너스
